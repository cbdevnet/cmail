== Webadmin ==

This is the web administration tool for cmail services.

== Setup ==

=== Requirements ===

* running cmail-api (see cmail-api setup guide)
* HTTP server to get the code

=== Setup ===

1. Copy code in the server path
2. Copy/move the config.js.sample file (found in static/js/) to config.js
3. Setup the api path/url in config.js
